This project is a **Sudoku Solver** implemented in **Java** using **recursion** and **backtracking**. The program can solve any 9x9 Sudoku puzzle provided by the user.

---

## Features

- Solve any valid 9x9 Sudoku puzzle.
- Accepts user input (`0` represents empty cells).
- Displays both the original and solved Sudoku in a clear 3x3 grid format.
- Implements backtracking algorithm for efficient solving.

---

## How to Use
1. Clone the repository or download the code.
2. Compile the Java file: `javac Sudokucode.java`
3. Run the program: `java Sudokucode`
4. Enter your Sudoku puzzle (use 0 for empty cells).
5. The program will display the solved puzzle if a solution exists.
